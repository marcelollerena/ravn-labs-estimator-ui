# Tailwind Color Configuration

## Color Choices

- **Primary:** `blue` — Used for buttons, links, key accents, active nav items
- **Secondary:** `slate` — Used for tags, highlights, secondary elements, industry badges
- **Neutral:** `zinc` — Used for backgrounds, text, borders, sidebar

## Usage Examples

Primary button: `bg-blue-500 hover:bg-blue-600 text-white dark:bg-blue-600 dark:hover:bg-blue-500`
Secondary badge: `bg-slate-200/60 text-slate-600 dark:bg-slate-700/50 dark:text-slate-400`
Neutral text: `text-zinc-600 dark:text-zinc-400`
Sidebar background: `bg-zinc-950`
Active nav: `bg-blue-500/10 text-blue-400`
Card border: `border-zinc-200 dark:border-zinc-700`
